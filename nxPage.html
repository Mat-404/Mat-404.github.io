Siemens NX Modeling

Video-guided practice

One of the classes I took at Florida Tech was called Aerospace Practicum.
This class contained some lessons in modelling parts such as brackets and washers using the Siemens NX software.
In my free time, however, I wanted to practice more than I was able to with the class.
Because of this, I took some time and tried to model the first 17 parts from a youtube video providing examples of practice parts.

Arduino Uno Case

Also as a part of Aerospace Practicum, we had some projects using the Arduino Uno. As an additional project, I purchased a small LCD screen to affix on top.
On top of this, I wanted to create a case for it so I could display it like an alarm clock.
I inevitably had issues with getting it printed through my local library, but I still created the models below in NX.
