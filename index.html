Hi! I am Mathieu, a current Senior studying Aerospace Engineering!
I created this page in order to showcase some of the more technical work I have done!

Projects:
SPARCC 3.0 (2024-2025)
Aerospace Engineering senior capstone project which seeks to create a solid pellet-fueled propulsion system for cubesats that fits within a 2U volume.
I am serving as the Systems Engineer for a team of seven to ensure each subsystem's design options will work with each other to produce the intended result.
Spacecraft Attitude Simulator (2024)
A MATLAB / Simulink class project which modeled the attitude dynamics and kinematics. Used modeled magnetorquers and reaction wheels alongside modeled sensors in order to detumble and maintain a nadir attitude.
Rocket Simulation (2023)
A program written in Python to determine the trajectory of a model rocket using specific thrust vs time data provided by Estes. Uses vector math to determine the maximum altitude and distance downrange it would land, assuming no outside wind.
Orbit Simulation (2022-2023)
A series of two projects in which I create numerical 2-D orbit simulations via Java and Python.
The first method calculates the net gravitational force on each body at each time step, so it can numerically model multiple bodies' interations
The second method uses Keplerian equations of orbital motion (ignoring inclinations) and can model multiple bodies analytically.
