SPARCC 3.0
In a nutshell, SPARCC 3.0 is an Aerospace Engineering Senior Design capstone project. This is a project which I act as Systems Engineer for. The aim of the project is to develop a solid propellant thruster for space use which fits inside a 2 U (2 10x10x10 cm cube) volume. This site aims to detail the major technical work I have done to serve this project.
